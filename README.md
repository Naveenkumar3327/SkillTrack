## 📖 Overview

**SkillTrack** is an AI-powered Skill Development & Employability Enhancement Platform designed to bridge the gap between academic learning and industry expectations.

The platform enables students to assess their skills, receive personalized learning roadmaps, build ATS-friendly resumes, prepare for interviews, practice coding, earn certifications, and improve placement readiness through AI-driven recommendations and analytics.

It also provides dedicated dashboards for Staff, Administrators, and Recruiters to efficiently manage students, learning content, assessments, and campus placements.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/SkillTrack.git
```

## Navigate

```bash
cd SkillTrack
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
